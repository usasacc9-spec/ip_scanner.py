# IP Scanner Tool

A simple multi-threaded IP scanner written in Python.

## Features
- Fast scanning using threads
- Cross-platform (Linux, Termux, Windows)
- Easy to use CLI interface

## Installation

Install Python first.

## Usage

Run the script:

python ip_scanner.py

Enter base IP like:
192.168.1

## Example Output

[+] 192.168.1.1 is ACTIVE
[+] 192.168.1.5 is ACTIVE

## Disclaimer

Use only on your own network. Unauthorized scanning is illegal.
